strcpy两种写法的看法

Heysion

2011-11-08 09:11

源代码直接Copy 见谅 据说这个是GNU的写法（my_strcpy2）

 1  char *
 2  my_strcpy2 (dest, src)
 3       char *dest;
 4       const char *src;
 5  {
 6    register char c;
 7    char * s = (char *)src;
 8    const int off = dest - s - 1;
 9    do{
10        c = *s++;
11        s[off] = c;
12      }while (c != '\0');
13    return dest;
14  }

另外一种写法来自GFree_Wind（my_strcpy1）

 1  char* my_strcpy1(char *dest, const char *src)
 2  {
 3      char *d = dest;
 4      register char c;
 5      do{
 6          c = *src++;
 7          *d++ = c;
 8      } while ('\0' != c);
 9      return dest;
10  }
